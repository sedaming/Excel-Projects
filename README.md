## 📌 Project Overview

This project involves a full end-to-end data analysis workflow using **Microsoft Excel**. Starting from raw data, I cleaned, transformed, and analyzed the [bike buyers dataset](https://github.com/AlexTheAnalyst/Excel-Tutorial) to uncover insights about customer demographics, purchasing behaviour, and retail sales performance.

The final output is an **interactive Excel dashboard** featuring KPI cards, dynamic charts, and slicers that allow users to filter and explore the data in real time.

---

## 🛠️ Tools & Techniques Used

- **Data Cleaning**: Removed duplicates, standardized categorical values, handled blank fields
- **PivotTables**: Aggregated data by demographics, region, and purchase behaviour
- **Slicers**: Dynamic filtering by marital status, region, and education
- **Charts**: Bar charts, line graphs, and clustered column charts
- **KPI Cards**: Highlighted key metrics (total buyers, avg income, commute trends)
- **Dynamic Reporting**: Dashboard auto-updates as filters change

## 🔍 Key Findings

- Customers with **middle income ranges** were most likely to purchase a bike
- **Short commute distances (0–1 miles)** showed the highest purchase rate
- **Male customers** purchased bikes at a slightly higher rate across most regions
- The **Middle Age group (31–54)** represented the largest buyer segment
- **North America** had the highest concentration of buyers in the dataset

## 🚀 How to View

1. Download the `.xlsx` file from the `Retail-Sales-Analysis/` folder
2. Open in **Microsoft Excel 2016 or later**
3. Navigate to the **Dashboard** tab
4. Use the **slicers** on the left to filter by Marital Status, Region, or Education
